# hsx-http

http api for tplink hs1xx smart plugs

## usage

fix dependencies
```
npm install
```

create config.json object in root folder
```
{
  "ip": "HS1XX_IP"
}
```

start server
```
npm start
```

## commands

```
http:/localhost/8010/on
http:/localhost/8010/off
http:/localhost/8010/toggle
```
